# Usage

Simple python script that when given a name will automatically copy comment header to clipboard.

![screenshot](./images/ss1.png)

## 1. V1

```
python3 codeCommentor.py -v1
```

Output:

```
/***********************
 * ----- COMMENT ----- *
 ***********************/
```

## 2. V2

```
python3 codeCommentor.py -v2
```

Output:

```
/*
 ***********************
 * ----- COMMENT ----- *
 ***********************
 */
```
